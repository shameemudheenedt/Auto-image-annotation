# Auto-image-annotation
Automating image annotations for object detection task
build a module in python using SAM(Segement Anything Model) pre trained model and pre trained model file to build auto annotation module.
Features

- Object detection using YOLO
- Automatic segmentation using SAM
- Saves binary masks
- Generates YOLO segmentation labels
- Saves visualization images
- Batch processing

Directory Structure
```
project/
│
├── auto_annotate.py
├── images/
├── labels/
├── masks/
├── visualizations/
└── weights/
```
